Welcome. To get a feel for Amber, follow along this quick tutorial.

Implementation Note

The sequence of events will be:

  1. Create a model (App.Person)
  2. Create an Array (App.people)
  3. Create a view and append it
  4. Get a shell, add yourself to the Array → watch the right side update live
  5. Add someone else to the Array → watch the right side update live
  6. Modify yourself → watch the right side update live
As the user moves through the steps, the copy above and button below will change.
Get Started
  • JavaScript
  • Templates
 // model
 App.Person = SC.Object.extend({
   firstName: null,
   lastName: null
 });

 // controller
 App.people = []

 // do:
 App.people.pushObject(App.Person.create({
   firstName: "Yehuda",
   lastName: "Katz"
 });

 <!-- view -->
 <ul>
   {{#each App.peopleController}}
     <li>{{firstName}} {{lastName}}</li>
   {{/each}}
 </ul>
